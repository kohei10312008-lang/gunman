📖 GUNMAN TYPER — 取扱説明書
🎮 ゲームの目的

画面に表示される単語をキーボードで素早く打つタイピングゲームです。

⌨️ 基本操作
A〜Z キー文字を入力する
Tab標的のフォーカス切替（SURVIVALのみ）
先頭文字その文字で始まる単語に自動切替
🎮 ゲームモード一覧
🎯 SURVIVAL10分間、奥から来る敵を撃ち抜く。スコア競争。
♾️ ENDLESS時間制限なし。HPが尽きるまで生き残れ。生存時間を競う。
⚡ TIME ATTACK固定問題を最速クリア！5/10/20/30問から選択。
🤠 DUEL1対1の早撃ち対決。何連勝できるか。
🎯 PRACTICE焦らず練習。五十音・濁音・単語カテゴリを選択。
🌐 言語モード
🔤 ENGLISH

英単語をアルファベットで入力。Lv1は1文字からスタート。Lv4以降は動物・食べ物・西部劇語もミックス。

🇯🇵 日本語

ひらがな表示→ローマ字で入力。
Lv10でローマ字が薄く、Lv11+で消える（かなのみモード）。

📈 難易度
EASYゆっくり・短い単語のみ。HP8。日本語はローマ字常時表示。
NORMAL標準スピード。HP7。日本語はLv10でうっすら→Lv11かなのみ。
HARD高速。HP4。日本語は最初からかなのみ。
🎯 SURVIVALとは

10分間（600秒）、奥から前進してくる敵シルエットをタイピングで撃ち抜くモード。スコアの高さを競います。

🕹️ ゲームの流れ
  1. 3・2・1カウントダウン後スタート
  2. 敵が奥から5レーンにランダム出現・前進
  3. 表示された単語を打つ → 撃破！
  4. 敵が手前に到達するとHPが減少
  5. HP0またはタイムアップでゲームオーバー
💯 スコアとコンボ
基本点文字数×10 ＋ レベル×5
コンボ×23連続撃破以上
コンボ×35連続撃破以上
📊 レベルアップと単語
Lv11文字（A, B…）/ か・き・く…
Lv22文字（GO, NO…）/ いぬ・ねこ…
Lv33文字（GUN, ACE…）/ はな・もり…
Lv4-54〜5文字＋動物・食べ物・自然語がミックス
Lv9+長単語（REVOLVER, DESPERADO など）
💀 ボス的

Lv5, 10, 15, 20… ごとにボス的が出現！
通常より大きく、複数の単語を連続で入力して倒す（HP制）。
撃破するとスコア×5ボーナス！

⏱ WPM・精度表示
WPM1分間のタイピング速度。60以上で緑・30以上で金。
ACC打鍵正確率。90%以上で緑・70%以上で金・69%以下で赤。
🤠 DUELとは

敵と1対1の早撃ち勝負！単語を制限時間内に打てれば勝利、時間切れで撃たれて終了。何連勝できるかを競います。

🕹️ 1戦の流れ
  1. 3・2・1 カウントダウン
  2. にらみ合い（0.8〜2.2秒のランダム待機）
  3. DRAW! と同時に単語表示
  4. 時間内に全文字を打つ → WIN! → 次の敵へ
  5. 時間切れ → 撃たれてゲーム終了
⏱️ 難易度別制限時間
EASY7秒スタート → 最短2.5秒
NORMAL5秒スタート → 最短1.8秒
HARD3.5秒スタート → 最短1.2秒 ＋ ミスで即死

日本語モードではLv10でローマ字が薄くなり、Lv11以降はかなのみで判断する力が必要になります。

⚡ TIME ATTACKとの違い

DUELは連勝数を競い、TIME ATTACKはクリアタイムを競います。
TIME ATTACKは5/10/20/30問から問題数を選択でき、問題数ごとに別ランキングが記録されます。

🏆 SURVIVAL攻略
  • 速さより正確さを意識してコンボを繋げよう
  • 先頭文字打鍵で短い単語から素早く片付ける
  • ゲージが赤くなる前に奥のうちに倒す
  • コンボ5以上でボーナス×3、連続撃破でスコアが急増
⚡ DUEL攻略
  • にらみ合い中は指をキーの上に置いて準備する
  • DRAW! の瞬間を見逃さず画面中央に集中
  • HARDはミスで即死。確実に1文字ずつ打つ
